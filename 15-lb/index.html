• window — глобальный объект браузера.
  • document — объект, представляющий HTML-документ.
  • console — объект для вывода информации в консоль браузера.

Описание объекта: const car = { make: 'Toyota', model: 'Corolla', year: 2020, color: 'red' };

Пример доступа к свойствам объекта в JavaScript: console.log(car.make); // Вывод: Toyota console.log(car['model']); // Вывод: Corolla

Команда для удаления свойства объекта: Для удаления свойства используется оператор delete. delete car.color; console.log(car.color); // Вывод: undefined

Команда для проверки существования свойства: Для проверки существования свойства используется оператор in или метод hasOwnProperty.

Команда для перебора свойств объекта: Для перебора свойств объекта можно использовать цикл for...in. for (let key in car) { if (car.hasOwnProperty(key)) { console.log(`${key}: ${car[key]}`); } }
